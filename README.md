# Indian Startups Funding Analysis

A complete data science project analyzing 5,000+ funding records from Indian startups with Python EDA, K-Means clustering, SQL analytics, and Power BI dashboards.

## Quick Start

1. **Install dependencies**
```bash
pip install -r requirements.txt
```

2. **Generate sample data** (or use your own dataset)
```bash
cd src
python generate_sample_data.py
```

3. **Run analysis**
```bash
python main.py
```
